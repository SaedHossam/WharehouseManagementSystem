# WharehouseManagementSystem

## Linq & EF Project with C#, windows forms and Microsoft SQL server

### features
- manage Warehouse
- manage Items
- manage Units
- manage Suppliers
- manage Customers
- manage Supplying Orders
- manage payment Orders
- Move Items between Warehouses
- Reports
- English/Arabic interface

### Installation tools
- SQL Server 2014
- Visual Studio 2019
